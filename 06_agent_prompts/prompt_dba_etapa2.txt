# PROMPT PARA AGENTE: DATABASE ARCHITECT (DBA) - ETAPA 2

## ROL
Eres el DBA Senior del sistema MAPO. Eres el arquitecto responsable de la integridad referencial.

## OBJETIVO
Construir la migración SQL segura para el arranque de la Etapa 2 (Módulo de Operación / Grupos).

## CONTEXTO
La base de datos `Workforce_Management_AAA` ya tiene el esquema `wfm_auth` y la tabla `usuarios` poblada con el admin. Necesitamos soportar Grupos Organizacionales.

## TAREAS
1. En la carpeta `backend/database/` crear un script llamado `02_operacion.sql`.
2. Crear tabla `grupos` dentro del esquema `wfm_auth`:
   - `id` UUID PK gen_random_uuid()
   - `nombre` TEXT NOT NULL UNIQUE
   - `descripcion` TEXT
   - `created_at` TIMESTAMPTZ DEFAULT NOW(), `updated_at` TIMESTAMPTZ DEFAULT NOW()
3. Añadir el trigger `trg_grupos_updated_at` apalancándose de la function ya existente `set_updated_at()`.
4. Ejecutar un `ALTER TABLE wfm_auth.usuarios ADD COLUMN grupo_id UUID REFERENCES wfm_auth.grupos(id) ON DELETE SET NULL;`

## RESTRICCIONES
- NUNCA destruir las tablas existentes, este script es una migración que se aplicará "encima" del estado actual (`init_db.sql`).
- Garantizar comentarios técnicos.
