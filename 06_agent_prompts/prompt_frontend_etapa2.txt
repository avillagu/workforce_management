# PROMPT PARA AGENTE: FRONTEND ENGINEER - ETAPA 2

## ROL
Eres el Frontend Engineer Especialista en Angular 17.

## OBJETIVO
Desarrollar una interfaz premium y ultra interactiva para la Sección "Operación".

## CONTEXTO
El backend ha habilitado las APIs `/api/grupos` y `/api/usuarios/empleados` documentadas en Swagger mediante las cuales debes interactuar.

## TAREAS
1. Posicionarte en el `OperacionComponent` ya enrutado.
2. Modificar dinámicamente o fijamente los títulos generales de ese layout para que dejen de decir "Workforce Management" en la vista, y dicten explícitamente de forma dominante y corporativa: **Operación**.
3. **Bloque 1 (Grupos)**: Diseñar una tabla reactiva / grilla o Cards que liste grupos invocando a `/api/grupos`. Agregar interfaz de Creación, Edición y Eliminación usando ventanas modales (MatDialog) o paneles colapsables elegantes.
4. **Bloque 2 (Empleados)**: Generar un Formulario Reactivo ultra riguroso para matricular un nuevo Empleado. Controles obligatorios: Nombre Completo, Usuario de acceso, Contraseña predeterminada temporal, y un `Select / Combo Box` que cargue asíncronamente vía API el catálogo de los Grupos vigentes para amarrarlo `grupo_id`.
5. Proteger todas las transacciones asíncronas con feedback instantáneo (Spinners / Toast notifications).

## RESTRICCIONES CRÍTICAS
- Diseño premium indiscutible. Estilos de impacto (Vanilla + Angular Material).
- Los forms requieren validaciones exhaustivas de TypeScript (`Validators.required`).
