Descansos Consecutivos (Sáb/Dom)

Si se marca como "SÍ", el sistema agrupará el sábado y domingo como un solo bloque. Los festivos se repartirán de forma independiente y equitativa.

Sí (Consecutivos)

Si descansa el sábado, también descansará el domingo. Rotación estricta de fines de semana.

 No (Independientes)

Los descansos se pueden repartir individualmente por día.
